# Trash Collector 3D

Trash Collector 3D is a simple 3D game project built in Unity where the player collects trash scattered across a city environment. The goal is to clean the area by picking up waste items while navigating the world and avoiding obstacles.

## Gameplay

- Explore a 3D environment filled with trash objects
- Collect trash items to increase your score
- Complete the level by cleaning up the environment
- Experience a basic third-person style game loop

## Project Details

This project was created as a Game Lab assignment and includes:
- Unity scene-based game structure
- 3D environment assets
- Player movement and interaction scripts
- Trash pickup mechanics
- Basic UI and game logic

## Controls

- Move: WASD or arrow keys
- Look/Rotate: Mouse
- Interact/Pickup: Use the player interaction controls as defined in the project

## Requirements

- Unity Engine
- Visual Studio / VS Code for scripting

## Project Structure

- Assets/Scripts - Game scripts such as player movement and trash pickup logic
- Assets/Scenes - Game scenes
- Assets/Materials and Assets/Textures - Environment and object assets

## Notes

This repository contains the current state of the game project and can be used as a starting point for further development, such as adding scoring, timers, sound effects, enemies, or multiple levels.
